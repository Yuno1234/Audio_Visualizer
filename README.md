# Audio_Visualizer

## General info
This project is a interactive audio visualizer using the p5.js.sound library.
	
## Technologies
Project is created with:
* HTML
* Javascript
* p5.js
* p5.js.sound
	
## Setup
1. Clone the github repositry onto desktop
2. Open the index.html file in the folder
